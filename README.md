# java-chess

체스 미션 저장소

# 기능 목록

- 기물
    - 종류를 가진다
    - 색을 가진다
    - 위치를 가진다
    - 이동할 수 있다
    - 다른 색의 기물을 공격할 수 있다
    - 다른 말을 뛰어넘을 수 없다(기본)
- 위치
    - file,rank 를 가진다
    - 이동할 수 있다
- 방향
    - 상하좌우
- file
    - a~h
    - 좌우로 이동 할 수 있다
        - 범위를 벗어난 이동은 예외를 던진다
- rank
    - 1~8
    - 위 아래로 이동 할 수 있다
        - 범위를 벗어난 이동은 예외를 던진다
